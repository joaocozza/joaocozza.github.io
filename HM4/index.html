PWA - Progressive Web Application


On Web developement, we have the methodology known as PWA. This feature combines different technologies in a way to make web application more stable and realiable. Below you will see some characteristics.


Main points of Progressive Web Design


Reliability:

This feature means that, when the user is using an mobile application, PWA loads the content of the application even if the user doesn't have a network.

Fasteness:

Today most people doesn't so much have patience to stay waiting for an web application to work. So, progessive web application comes to make sure that the user won't waint so much time to have al the content of the page loaded.

Cross-device capability:

PWA are built in programming languages such as: HTML, CSS and JavaScript that is hosted in web server, which is built for Web. However, all the content crated for web, also works in web broswers for mobile devices, considering that PWA, scales up the same content presented for desktop's and laptops, in application for mobile and broswers embbeded on smart TVs.
