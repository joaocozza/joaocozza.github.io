WURFL Capabilites


  1. The is_wireless-device will be set to false if it is a laptop or even if the laptop has wireless network card

  2. The xhmtl_make_phone_call_string can be used as a hint to start a phone call from a link or not and you can combine the value of that string with a phone number to make the link.

  3. The built_in_camera is to check the existence of a camera on the device, to allow the user to take a picture
